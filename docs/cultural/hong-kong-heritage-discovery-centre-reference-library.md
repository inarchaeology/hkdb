# 文物探知館參考圖書館

## 簡介

　　館藏香港歷史、考古學、博物館學、文物保護、建築學等參考資料。其中比較重要的資料是未發表的香港考古報告。已知館内收錄445份已發表和未發表的發掘和調查報告。
  
　　館内資料均不外借，借閲閉架資料前需填妥「借閲閉架參考資料申請表格」。一般情況下，每次可預約5份閉架資料。經電郵發送申請表格後，圖書館員會致電申請人，確認到館日期。圖書館提供掃描影印服務，以單份資料計，每人每次可掃描不多於總頁數的十分之一。
